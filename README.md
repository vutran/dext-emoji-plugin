> [Dext](https://github.com/vutran/dext) plugin to search for emojis.

![](screenshot.png?raw=true)

## Install

```bash
$ dpm install dext-emoji-plugin
```

*Requires [Dext](https://github.com/vutran/dext).*

## Usage

In Dext, type `emojis`, then your query.
